Side Navbars

Built-in support for responsive Side Navbars, enables to rapidly prototype different layout styles.

Side Navbars DO NOT require additional JS code. They're compatible with any implementation of Bootstrap

Side navbar

  • This is the main component, it always has the following classes: navbar navbar-side collapse.
  • According to your desing you'll need to use navbar-side-left or navbar-side-right, defining a side for your navbar.
  • Finally you should define a breaking point to display or hide your navbar: navbar-side-expand-{BREAKING-POINT}, where BREAKING-POINT is sm, md, lg, xl.

Sample Sidenavbar code

<nav class="navbar navbar-side navbar-side-left navbar-side-expand-lg navbar-dark bg-dark-alpha-2 collapse" id="sideNavbar">
  <a class="navbar-brand d-none d-lg-flex d-xl-flex bg-shadow-1" href="#">
    <span>YOUR_TITLE</span>
  </a>
  <ul class="navbar-nav">
    <li><a class="nav-link" href="#">Nowhere</a></li>
    <li><a class="nav-link" href="#">Another link(?)</a></li>
    <li><a class="nav-link" href="#">Stay here</a></li>
  </ul>
</nav>

Side Navbar Replacer

  • This is a small navbar that appears replacing the main navbar, when the breaking point goes lower than the specified value.
  • It always has the following classes: navbar navbar-side-replacer-{BREAKING-POINT}, where BREAKING-POINT is sm, md, lg, xl.
  • It should have a toggler button for your main side navbar.

Sample Sidenavbar replacer


  <a class="navbar-brand" href="#">
    <span>YOUR_TITLE</span>
  </a>
  <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#sideNavbar"
    aria-controls="navbarSupportedContent" aria-expanded="false" aria-label="Toggle navigation">
    <span class="navbar-toggler-icon"></span>
  </button>
</nav>

More info

  • Use the class m-navbar-side-left-{BREAK-POINT}, , where BREAKING-POINT is sm, md, lg, xl, to define a margin between your main content, while the side navbar is visible.
  • Side navbars are fully compatible with Bootstrap color utilities and base navbars.

General structure sample

<body>
  <!-- Side navbar -->  
  <nav class="navbar navbar-side navbar-side-left navbar-side-expand-lg collapse">...</nav>

  <!-- Replacer -->  
  <nav class="navbar sticky-top navbar-side-replacer-lg">...</nav>
  
  <!-- Your main content -->
  <div class="m-navbar-side-left-lg">
    <div class="container">
      ....
    </div>
  </div>
</body>
