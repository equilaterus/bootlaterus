Works with almost any JS implementation of Bootstrap.

Bootlaterus doesn't require additional JS files to work, so it is fully compatible with Bootstrap Default jQuery implementation but also with ports like React-Bootstrap, ReactStrap, ng-bootstrap, bootstrap+vuejs and almost any JS implementation of Bootstrap Components.

What is Bootlaterus

Bootlaterus is Compatible and Easy to use Bootstrap Superset.

It includes Bootstrap v4 and has extended components that don't require additional Javascript to work.

We provide custom templates for general use applications like stores or social networks and also for Wiki-like pages, GitHub Pages or documentation-sites.

Based on Bootstrap v4.

Quick start

CDN

Add Bootlaterus that already includes bootstrap CSS (to see additional themes go here): 

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootlaterus@latest/dist/css/bootlaterus-cfonts.min.css" crossorigin="anonymous">

Include additional javascript libraries (JS, Popper, and jQuery):

<script src="https://code.jquery.com/jquery-3.3.1.slim.min.js" integrity="sha384-q8i/X+965DzO0rT7abK41JStQIAqVgRVzpbzo5smXKp4YfRvH+8abtTE1Pi6jizo" crossorigin="anonymous"></script><script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.14.7/umd/popper.min.js" integrity="sha384-UO2eT0CpHqdSJQ6hJty5KVphtPhzWj9WO1clHTMGa3JDZwrnQq4sF86dIHNDz0W1" crossorigin="anonymous"></script><script src="https://stackpath.bootstrapcdn.com/bootstrap/4.3.1/js/bootstrap.min.js" integrity="sha384-JjSmVgyd0p3pXB1rRibZUAYoIIy6OrQ6VrjIEaFf/nJGzIxFDsf4x0xIM+B07jRM" crossorigin="anonymous"></script>

NPM

You can install bootlaterus using NPM:

npm install bootlaterus --save
